# Floorball_stats
This project is supposed to provide an easy way to track the stats of my floorball team during a whole season of 20 games. This is nothing fancy, simply an alternative to using something boring like excel.

## Installation 
To run the server make sure to have python3 installed on the system. Then the backend can be installed and setup using the following commands:

git clone git@github.com:EricL03/Floorball_stats.git 
´´´
